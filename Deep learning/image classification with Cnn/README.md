# CIFAR-10 Image Classification using CNN

## Description
This project implements a Convolutional Neural Network (CNN) to classify images from the CIFAR-10 dataset into 10 different categories. The model is built using TensorFlow and Keras, with image preprocessing, training, evaluation, confusion matrix visualization, and prediction on test images.

## Features
- Image classification using CNN
- Data preprocessing and normalization
- Model training and evaluation
- Confusion matrix visualization
- Single and batch image prediction
- Model saving and loading

## Technologies Used
- Python
- TensorFlow / Keras
- NumPy
- Matplotlib
- Scikit-learn

## Dataset
CIFAR-10 Dataset
