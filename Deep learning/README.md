# Fuzzy Deep Learning for Face Classification

## Description
This project implements a hybrid Fuzzy Deep Learning model for face image classification using the LFW (Labeled Faces in the Wild) dataset. It combines Convolutional Neural Networks (CNNs) with fuzzy logic to improve classification by handling uncertainty in predictions.

## Features
- Face image classification using CNN
- Fuzzy logic integration
- Image preprocessing and normalization
- Model training and evaluation
- Accuracy, loss, and confusion matrix visualization
- Model saving for future use

## Technologies Used
- Python
- TensorFlow / Keras
- NumPy
- Matplotlib
- Scikit-Fuzzy
- KaggleHub

## Dataset
LFW (Labeled Faces in the Wild)
