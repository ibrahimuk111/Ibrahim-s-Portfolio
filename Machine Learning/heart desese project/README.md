# Heart Disease Prediction using Logistic Regression

## Description
This project predicts the likelihood of heart disease using a Logistic Regression model. It includes data preprocessing, exploratory data analysis (EDA), feature engineering, model training, and performance evaluation using various classification metrics and visualizations.

## Features
- Data preprocessing and feature scaling
- Exploratory Data Analysis (EDA)
- Logistic Regression model
- Model evaluation (Accuracy, Precision, Recall, F1-Score, ROC-AUC)
- ROC Curve and Feature Importance visualization

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Dataset
Heart Disease Dataset
