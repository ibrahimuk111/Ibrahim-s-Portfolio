# Customer Churn Prediction using Machine Learning

## Description
This project predicts customer churn using machine learning techniques on the Telco Customer Churn dataset. It includes data preprocessing, feature encoding, model training, evaluation, and comparison of multiple classification algorithms to identify customers who are likely to leave a service.

## Features
- Data cleaning and preprocessing
- Missing value handling
- Feature encoding
- Logistic Regression, Random Forest, and XGBoost models
- Model performance comparison
- Confusion matrix and classification report
- Model saving using Joblib

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- XGBoost
- Matplotlib
- Seaborn
- Joblib

## Dataset
Telco Customer Churn Dataset
